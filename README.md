### Installation
##### Dependencies
        - Nodejs v6+
        - Npm 
        - Android SDK
        
##### Setup
        - sudo npm install -g gulp bower cordova ionic
        - sudo npm install
        - bower update
        - ionic platform add android
        - ionic platform add ios
        
##### Run
        - ionic serve
        
##### Build Android
        - ionic build android
        
##### Build IOS
        - ionic build ios
